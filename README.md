 Clinic-Appointment-System
University Clinic Appointment System – SDG 3 Good Health and Well-Being.
The system allows students to book, cancel, and view appointments, while enabling doctors and admins to manage schedules efficiently.

  Overview of SDLC Work
This project follows a structured SDLC process:
1. **Planning**: Defined the product vision, created user stories, and set up the project structure.
2. **Design**: Created UML diagrams (Use Case and Class diagrams) to model the system and planned the UI.
3. **Testing**: Developed test plans and user testing feedback to ensure quality and improvement.
4. **Final**: Finalized the prototype, prepared documentation (report), and showcased the results.

  Features
- **Register**: Students can create an account to book appointments.
- **Login**: Registered users can log in to manage their appointments.
- **Book Appointment**: Students can select a doctor, date, and time.
- **Cancel Appointment**: Students can cancel previously booked appointments.
- **Admin Functionality**: Admins can manage doctor schedules.

  SDG 3: This system improves access to healthcare for students by reducing queues and simplifying appointment management.
